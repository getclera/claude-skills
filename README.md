# Claude Skills by Clera

Open-source [Claude skills](https://support.anthropic.com/en/articles/11147069-what-are-skills) for hiring and career prep. Built by [Clera](https://getclera.com) — the AI-native hiring platform for startups.

## Skill Bundles

### [Recruitment](./recruitment/) — for hiring managers & recruiters

10 skills covering the full hiring loop: writing JDs, screening CVs, scoring interviews, building rubrics, candidate outreach, offers, rejections, intake briefs, and competitor talent intelligence.

**[Download recruitment-skills.zip](https://github.com/getclera/claude-skills/releases/latest/download/recruitment-skills.zip)**

### [Interview Prep](./interview/) — for job seekers

10 skills for finding the right roles and winning them: research, prep, positioning, negotiation, and decisions. Not about auto-applying — about giving you an information edge.

**[Download interview-skills.zip](https://github.com/getclera/claude-skills/releases/latest/download/interview-skills.zip)**

## How to Install

1. **Enable skills.** Claude.ai → Settings → Capabilities → toggle on "Code execution and file creation."
2. **Download a bundle** from the links above (or from [Releases](https://github.com/getclera/claude-skills/releases)).
3. **Upload the `.zip`** in Claude.ai → Customize → Skills → **+** → Upload a skill.
4. **Toggle each skill on.** They arrive switched off.

Each bundle's README has detailed setup instructions, including optional MCP connectors that power three of the skills.
